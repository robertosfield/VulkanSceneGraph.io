Detailed Description

Builder class that creates subgraphs that can render primitive geometries. Supported shapes are Box, Capsule, Cone, Cylinder, Disk, Quad, Sphere and HeightField. Uses GeometryInfo and StateInfo to guide the geometry position/size and rendering state.
